# Anki Template Designer Configuration

## Settings

- **preview_width**: Width of preview window in pixels (default: 800)
- **preview_height**: Height of preview window in pixels (default: 600)
- **default_platform**: Default platform to show ("desktop" or "ankidroid")
- **ankidroid_theme**: AnkiDroid theme for preview ("light" or "dark")
- **show_both_platforms**: Show both desktop and AnkiDroid previews side-by-side
- **auto_refresh**: Automatically refresh preview when template changes
